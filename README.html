# Gesture Controlled Music Player This is a music player built with [electronJS](https://www.electronjs.org/) and [handsfree.js](https://handsfree.js.org/) in which you can play, pause, skip and change the volume of music, using just hand gestures. # Requirements * A working webcam with good quality output. * A Windows PC (macOS not supported yet). * Speakers on your PC # Installing There is currently no MSI or EXE to install the program. You need to package, mannually extract the program and create the necessary shortcuts.
